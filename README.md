# EXERCISES
Assignment 1
Δημιουργήθηκε πρόγραμμα το οποίο θα δέχεται έναν ακέραιο Ν. 
Το πρόγραμμα θα εμφανίζει στην οθόνη: min max όπου min (max) το μικρότερο (μεγαλύτερο) ψηφίο του αριθμού N
E.g.
Για είσοδο 5378 το πρόγραμμα θα πρέπει να εμφανίζει:
3 8


Assignment 2
Δημιουργήθηκε πρόγραμμα που διαβάζει έναν αριθμό 4 ψηφίων και αντιστρέφει το πρώτο και το τελευταίο ψηφίο του. 
Π.χ. Αν ο αριθμός είναι ο 1289 θα πρέπει να εκτυπώνει 9281.
